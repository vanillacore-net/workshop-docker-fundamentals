Willkommen zum Docker Workshop

Diese Webseite läuft in einem selbst erstellten Docker-Container und dient als Referenz für die wichtigsten Docker Befehle.

Einfach

Mit nur wenigen Befehlen können wir Container erstellen und verwalten.

Portabel

Container können auf verschiedenen Umgebungen identisch ausgeführt werden.

Isoliert

Jeder Container läuft in seiner eigenen Umgebung.

Docker Befehls-Referenz

Container Management

docker run IMAGE Container starten
docker ps Laufende Container anzeigen
docker ps -a Alle Container anzeigen (auch beendete)
docker exec -it CONTAINER BEFEHL Befehl in laufendem Container ausführen
docker stop CONTAINER Container anhalten
docker rm CONTAINER Container löschen

Image Management

docker images Verfügbare Images anzeigen
docker build -t NAME . Image aus Dockerfile erstellen
docker rmi IMAGE Image löschen
docker tag IMAGE:TAG NEUER_NAME:TAG Image umbenennen/taggen

Volume & Netzwerk

docker volume create NAME Volume erstellen
docker volume ls Volumes auflisten
docker run -v VOLUME:PFAD IMAGE Container mit Volume starten
docker run -p HOST:CONTAINER IMAGE Port-Mapping einrichten

Docker Hub

docker login Bei Docker Hub anmelden
docker push USER/IMAGE:TAG Image zu Docker Hub pushen
docker pull IMAGE:TAG Image von Docker Hub holen
